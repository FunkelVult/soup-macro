Soup Macro

Leistungsstarkes Macro-Tool mit Spam, Tame-Automatisierung und eigenem Makro-Builder. Kostenlos, kein Python nötig.

Download — SoupMacro.exe
Windows · Kein Python nötig · Startet als Admin

Features

⚡

Spam Modus

Drückt eine Taste mit einstellbarem Intervall — bis auf 1ms genau. Perfekt für schnelle Wiederholungen.

🎯

Tame Modus

Automatische Tame-Sequenz: Taste 2 drücken, warten, Taste 1 drücken, warten — endlos wiederholen.

🛠️

Eigene Makros

Baue beliebige Tastenkombinationen mit Wartezeiten. Speichern und laden als .json Datei.

🔄

Auto-Update

Das Programm prüft beim Start automatisch ob eine neue Version verfügbar ist und installiert sie mit einem Klick.

🔑

Hotkeys

Starte und stoppe alle Modi mit F1–F3 und F12, ohne das Fenster zu wechseln.

🎨

Modernes UI

Dunkles Design mit Live-Countdown, Fortschrittsbalken und farbkodierten Status-Anzeigen.

Installation

1

SoupMacro.exe herunterladen Klicke oben auf den Download-Button

2

Als Administrator starten Windows fragt automatisch nach — einfach bestätigen

3

Loslegen Modus wählen, Werte anpassen, starten
